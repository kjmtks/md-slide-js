## Features
* 独自拡張した Markdown でスライドを記述できます
* 作成したスライドのスライドショーを実行したり，PDFとして出力できます
* ローカルファイルも使用できます
* ブラウザだけで動作します
* PDF として出力することができます
* 発表者ツールが利用可能です
